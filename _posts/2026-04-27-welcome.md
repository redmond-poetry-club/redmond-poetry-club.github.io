This is your first post! Edit or delete this file and create new posts in the `_posts` folder.

## How to add new posts

Create a new file in `_posts/` with this naming format:
```
YYYY-MM-DD-title-of-your-post.md
```

Each post needs front matter at the top:
```yaml
---
layout: post
title: "Your Post Title"
date: 2026-04-27
---
```

Then write your content in Markdown below the front matter.
